## 程序的作用
Chpy可以允许你使用中文指令和中文翻译库进行编程,而中文翻译库可以自己编辑
## 怎么编译.poty文件
.poty可以通过使用Chpy构建指令来翻译成.py文件:
py pot.py -build <输入文件名> <输出文件名> [可选指令]
### 可选指令:
-jump_err:遇到错误继续编译
-run:编译完后启动
